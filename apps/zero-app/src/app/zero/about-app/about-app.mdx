### Save your Data

- **Lost** - Unfairly closed/banned account resulting in data loss.
- **Blocked** - Paywalling/Shutdown of an app with your data.
- **Tampered** - Images/Music/Videos removed from your library/playlist without permission.
- **Scattered** - Searching for specific data across multiple libraries
- **Cluttered** - Managing a huge list of user data in your own files.
- **Hassle to Backup** - Can't export or manually exporting multiple sources.
- **Unheard** - Complaints, Suggestions falling on deaf ears.
- **Paranoia** - Can't sleep knowing any of the above can happen at any time.

### Use this App

- **Free to Use** - Completely free aside from cloud-specific features.
- **Offline Support** - Most features work offline unless requiring cloud resources.
- **Cross Platform** - Supports web, mobile and desktop.
- **Agnostic Compatibility** - Not dependent on any one vendor.
- **Requests** - Team is open to requests/suggestions/complaints on any of our projects.
- **Open Source Forever** - Transparent and forkable source code, so long as there are contributors this won't end up in a project graveyard. (AGPL-3.0)
- **Community** - Join our Community and let's talk.

### Centralize

- **All your Data in One Place** - One library to manage your notes, images, music, videos, etc.
- **Bookmark External Data** - Extract and Save data from third party sources.
- **Actual Universal ID Registry** - Generate unique IDs for each and every piece of data.
- **One Compiled File** - Easier to manage, backup and secure one file. large size contents are stored separately.
- **Multiple Users** - Multiple users can share the same data vault at the same time.
- **Full User Statistics** - Generate info otherwise impossible with distributed user data.

### Control

- **User Controlled Data** - No vendor locking, no unauthorized tampering of user data.
- **Exportable** - Freely move data from this app in standard formats.
- **File Storage** - Store your data in your choice of cloud storage. (multiple cloud syncing)
- **User Encryption** - Open source zero trust E2E data encryption. (general/specific encryption)
- **Auth Protocols** - App authentication. (Lockout Prevention, 2FA)
- **Advanced Security Features** - Authless features, encrypted data sharing.
- **Fine User Access Control** - Authorize individual, group, change requests.

### Manage

- **Advanced Search** - Search with advanced filtering and sorting. (layered queries, inclusion/exclusion, random sorting)
- **Tag System** - Customize your data tags. (tag aliases, merged group tags, metatags, ai tagging, public/personal tags)
- **History Tracking** - Optionally tracks user browsing. (latest viewed, oldest viewed, most viewed, least viewed, last added, last modified)
- **Changes Tracking** - Tracks all data modifications.
- **Remote Storage** - Control all user data in multiple cloud storages.
- **3rd Party Extensions** - Supports 3rd party plugins and external data seeds.
